# EasyChatbot4Beginners

## python 초보자를 위한 Chatbot

- 목표

    - 초보자가 학습할 때는 정확한 내용을 학습하는 것이 중요합니다. 그러나 일부 블로그의 내용은 정확하지 않을 수 있습니다. 
    이 ChatBot은 인증된 소스를 참조하여 검색 시간을 줄이고 답변의 정확성을 높이기 위해 설계되었습니다.
